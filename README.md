# Ruby-on-Rails
# RUBY 
Ruby is an object-oriented programming language 
Everything in Ruby is an object 
Ruby follows SOLID principles

# Ruby on Rails
Ruby on Rails is a framework of Ruby 
It is used for web application development 
Rails follows MVC architecture 
The latest stable version of Ruby on Rails is Rails 8.0. 

# Rails is a Framework of Ruby 
Ruby → Programming Language
Rails → Web Framework built using Ruby
Rails cannot work without Ruby 

# Day 1 Setup Guide 
**Installing Ruby** 
Step 1: Open Browser 
Step 2: Visit Ruby Official Website
  Go to the Ruby official website 
  Navigate to the Downloads section
Step 3: Download Ruby 
  Ruby 4.0.0 with Devkit 
Step 4: Install Ruby 
  Complete the installation process 
Step 5: Verify Ruby Installation 
  open terminal and run 
    ruby -version 
    
**Installing Ruby on Rails** 
Step 1: Install Rails 
  open terminal and write cammond 
    gems install rails 
      if (error) {open terminalas -> run as administator } 
Step 2: Verify Rails Installation 
  rails -v
  
**Creating a New Rails First Application** 
Step 1: Create a Ruby Folder in Desktop 
Step 2: Open Terminal / CMD 
  cd FoldeName
Step 3:Create a New Rails App 
  rails new myApp 
Step 4: Move into Application Folder 
  cd myApp
Step 5 : Open Application in Browser
  localhost:3000
