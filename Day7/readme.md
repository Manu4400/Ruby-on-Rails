<h2>Class</h2> 
<ul> <li>Class names should follow <b>CamelCase</b></li> <li>First letter must be capitalized</li> </ul> <p><b>Example:</b></p> <pre> class CustomController end </pre> <hr> <h2>Methods</h2> <ul> <li>Method and variable names should follow <b>snake_case</b></li> <li>All letters should be lowercase</li> </ul> <p><b>Example:</b></p> <pre> def limit_active end </pre> <hr> <h2>Inheritance</h2> <ul> <li>Ruby supports <b>single inheritance</b></li> <li>The <b>super</b> keyword is used to access parent class methods or variables</li> </ul> <p><b>Method Lookup Hierarchy:</b></p> <pre> Child Class ↓ Base Class ↓ Object ↓ Kernel (module) ↓ BasicObject </pre> <hr> <h2>Ancestors</h2> <ul> <li>Ruby searches methods starting from the child class</li> <li>If not found, it continues through the ancestor chain</li> </ul> <p><b>Example:</b></p> <pre> Child → Parent → Object → Kernel → BasicObject </pre> <hr> <h2>respond_to?</h2> <ul> <li>Used to check whether an object responds to a method</li> <li>Returns <b>true</b> or <b>false</b></li> </ul> <p><b>Example:</b></p> <pre> "Sample".respond_to?(:upcase) </pre> <p> <b>Note:</b> <code>respond_to?</code> also returns true for inherited methods from Object, Kernel, or BasicObject if they are not overridden. </p> <hr> <h2>Important Notes</h2> <ul> <li><code>puts</code> is a <b>private method</b> from the Kernel module</li> <li>Private methods cannot be called with an explicit receiver</li> <li><code>BasicObject</code> is the root class in Ruby</li> <li>It contains only minimal methods and supports private behavior</li> </ul>
